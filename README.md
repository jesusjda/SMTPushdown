Installation
============

Prerequisites
-------------
-  OCaml (Version 4.0 verified to work) [On Debian derivatives: apt-get install ocaml]
-  Sexplib (Version 7 verified to work)  [On Debian derivatives: apt-get install libsexplib-camlp4-dev]

Building
--------
Run "make".

Usage
=====

Consult "convert.native --help".
